# Support

## Where to ask questions

- Use GitHub Discussions for setup questions, configuration help, and general usage.

## Where to report bugs

- Use GitHub Issues for reproducible bugs.
- Include: image tag/version, host OS/arch, steps to reproduce, and relevant logs.

## Feature requests

- Use GitHub Issues for feature requests. Please describe the problem/use case and what success looks like.

## Security

- Do not report vulnerabilities via public issues.
- See `SECURITY.md` for the preferred reporting process.
